- git init -> đăng ký một folder trở thành một folder làm việc với git. (tạo mới folder .git)

- git remote add origin https://duong-dan-den-thu-muc-git -> cho phép thư mục hiện tại làm việc với một thư mục git ở xa (server git) với tên origin.
Đăng ký url https://duong-dan-den-thu-muc-git với tên gọi là origin.

- git status -> kiểm tra trạng thái các file trong thư mục git.

- git config --global user.email "youremail@gmail.com" -> Đăng ký email với thư mục git. Thường chạy khi lần đầu làm việc với git.
- git config --global user.name "your name" -> Đăng ký name với thư mục git. Thường chạy khi lần đầu làm việc với git.

- git add tenfile tenthumuc -> "Đăng ký những file, thư mục để chuẩn bị đưa lên server git".
- git add . để push cả thư mục lên 
- git commit -m "Tin nhắn" -> "Dán nhãn cho những file vừa được đăng ký đưa lên git."
- git push origin master -> "Đưa những file vừa đăng ký và dán nhãn lên thư mục git (tên viết tắt là origin)"
